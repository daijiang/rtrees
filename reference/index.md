# Package index

## Main function

- [`get_tree()`](https://daijiang.github.io/rtrees/reference/get_tree.md)
  : Get one or multiple trees from megatree(s)

## Other helper functions

- [`sp_list_df()`](https://daijiang.github.io/rtrees/reference/sp_list_df.md)
  : Convert a vector of species names to a data frame
- [`add_root_info()`](https://daijiang.github.io/rtrees/reference/add_root_info.md)
  : Add genus and family basal/root node information to a phylogeny
- [`get_graft_status()`](https://daijiang.github.io/rtrees/reference/get_graft_status.md)
  : Extract grafting status information as a data frame
- [`get_one_tree()`](https://daijiang.github.io/rtrees/reference/get_one_tree.md)
  : Derive a phylogeny from a mega-tree
- [`rm_stars()`](https://daijiang.github.io/rtrees/reference/rm_stars.md)
  : Remove trailing \*
- [`bind_tip()`](https://daijiang.github.io/rtrees/reference/bind_tip.md)
  : Bind a tip to a phylogeny
- [`bind_tip_df()`](https://daijiang.github.io/rtrees/reference/bind_tip_df.md)
  : Bind a tip to a phylogeny (data frame version)

## Taxonomic data

  : Taxonomic groups supported
- [`classifications`](https://daijiang.github.io/rtrees/reference/classifications.md)
